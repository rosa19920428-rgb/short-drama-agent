# 核心期待（钩子）系统完整指南

## 🎯 核心期待的作用

核心期待是推动剧情和情感发展的双引擎，确保短剧有持续的张力：

### 外在期待（External Hook）
- **定义**：推动剧情发展的关键悬念
- **示例**：
  - "契约婚姻何时被揭穿"
  - "男主的隐藏身份何时曝光"
  - "幕后黑手何时现身"
  - "复仇计划何时收网"

### 内在期待（Internal Hook）
- **定义**：角色关系发展的情感期待
- **示例**：
  - "男主何时放下骄傲真诚表白"
  - "女主何时相信自己配得上男主"
  - "两人何时打破身份悬殊真正相爱"
  - "信任破裂后如何重建"

## 📝 创意阶段

### AI自动生成
当AI生成5个创意时，会自动包含：
- ✅ 外在期待（基于创意类型智能生成）
- ✅ 内在期待（基于情感线智能生成）

**示例创意结构**：
```
标题：The Billionaire's Contract Bride
概念：平凡设计师助理与冷酷地产大亨的契约婚姻
梗概：500字详细故事...
外在期待：契约婚姻何时被揭穿，男主的真实目的何时暴露
内在期待：女主何时不再自卑，男主何时真心爱上而非利用
```

### 手动编辑
点击创意的"编辑"按钮，可以修改：
1. **标题**：英文标题
2. **核心设定**：一句话概念
3. **故事梗概**：400-500字详细描述
4. **外在期待**：剧情钩子（必须填写）
5. **内在期待**：情感钩子（必须填写）

**保存时验证**：
- 如果外在期待为空，会提示"请填写核心期待 - 外在事件"
- 保存后显示"✅ 创意已更新"和"✏️已编辑"标记

## 🎬 大纲阶段融入

### 如何融入剧情
生成大纲时，AI会收到明确的指令：

```
【核心期待 - 外在钩子】
契约婚姻何时被揭穿，男主的真实目的何时暴露

必须在剧情中体现此外在悬念，每集都要推进这个钩子。

【核心期待 - 内在钩子】
女主何时不再自卑，男主何时真心爱上而非利用

必须在情感线中体现此情感发展，让角色关系逐集递进。
```

### 每集体现方式
**外在钩子融入**：
- 第1集：埋下契约婚姻的伏笔（签订契约）
- 第5集：契约关系险些被第三人发现
- 第10集：重大危机，契约即将曝光
- 第15集：身份揭露，矛盾爆发
- 第20集：解决契约问题，达成真正关系

**内在钩子融入**：
- 第1-5集：女主自卑，男主高冷（陌生→好奇）
- 第6-10集：渐生好感，但心墙仍在（好奇→吸引）
- 第11-15集：情感升温，但身份悬殊带来阻碍（吸引→暧昧→矛盾）
- 第16-20集：突破心理障碍，真诚相爱（矛盾→和解→深爱）

## ✏️ 编辑核心期待

### 何时编辑
1. **AI生成的不够具体**：改为更精准的描述
2. **想改变剧情方向**：调整钩子引导大纲生成
3. **增加悬念层次**：让期待更引人入胜

### 编辑示例

**修改前**（太泛）：
- 外在期待：悬念待揭晓
- 内在期待：情感发展待展开

**修改后**（具体）：
- 外在期待："女主设计的作品何时被认可为男主公司关键项目"
- 内在期待："男主何时发现女主就是童年救命恩人"

## 🔄 工作流程

### 完整流程示例

1. **生成创意**
   - 点击"AI生成5个创意"
   - 检查每个创意的外在/内在期待

2. **选择创意**
   - 点击"选择"按钮选中一个创意
   - 查看下方"已选择的创意"区域

3. **编辑钩子（可选）**
   - 点击"编辑"按钮
   - 修改外在期待和内在期待，使其更具体
   - 点击"保存修改"
   - 确认显示"✏️已编辑"标记

4. **进入角色设计**
   - 点击"进入角色设计 →"
   - 创建角色

5. **生成标题**
   - 生成5个标题选项
   - 选择心仪标题

6. **生成大纲**
   - 点击"生成分集大纲"
   - **关键**：AI会读取核心期待，并在每集中融入
   - 检查大纲是否体现了外在和内在钩子

7. **生成剧本**
   - 基于大纲生成完整剧本
   - 剧本会延续核心期待的线索

## 🐛 故障排除

### 问题：外在/内在期待显示为"悬念待揭晓"
**原因**：AI没有生成具体的钩子
**解决**：
- 编辑创意，手动填写具体的钩子
- 或重新生成创意（每次生成都会不同）

### 问题：大纲没有体现核心期待
**原因**：大纲生成prompt没有正确使用钩子
**解决**：
- 检查ai-service.js的buildBatchPrompt函数
- 确认【核心期待 - 外在钩子】和【核心期待 - 内在钩子】部分已正确添加

### 问题：编辑后没有同步更新
**原因**：编辑的创意不是当前选中的
**解决**：
- 确保编辑前点击了"选择"按钮
- 或编辑后重新点击"选择"按钮

## 💡 最佳实践

1. **具体化**：避免"悬念待揭晓"，改为具体事件
2. **双重钩子**：外在钩子推动剧情，内在钩子深化情感
3. **可追踪性**：确保每一集都能追踪到钩子的推进
4. **层次递进**：钩子应该逐集加深，不是原地踏步
5. **编辑时机**：在生成大纲前确定钩子，大纲生成后不宜大幅修改
