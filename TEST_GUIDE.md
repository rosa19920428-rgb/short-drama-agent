## 测试步骤（按顺序）

### 第一步：保存并刷新
1. 按 `Ctrl+S` 保存 `js/ai-service.js`
2. 按 `Ctrl+S` 保存 `js/ui.js`
3. 浏览器按 `Ctrl+Shift+R` 强制刷新

### 第二步：测试人物生成
1. 选择类型（Female → Billionaire CEO）
2. **先生成一个创意**（必须！）
3. 进入角色设计步骤
4. 点击 **"🤖 AI生成全部角色"**
5. 等待10-15秒
6. **检查**：
   - [ ] 人物名字是欧美名（Sophia、Alexander等）
   - [ ] 每个人物卡片有 **"📖 查看详细小传"** 按钮
   - [ ] 点击按钮能看到详细的外貌、性格、背景、动机、秘密

### 第三步：测试大纲生成
1. 选择20集（先不要40集）
2. 点击"生成分集大纲"
3. **观察加载提示**：
   - 应该显示 "AI正在分批生成大纲 (1/4批，每批5集)..."
   - 然后是 "(2/4批)..." → "(3/4批)..." → "(4/4批)..."
4. **等待完成**（约40-60秒）
5. **检查**：
   - [ ] 所有20集都有内容，没有"content missing"
   - [ ] 每集都有完整的剧情、钩子、爽点
   - [ ] 场景是欧美地点（Manhattan、LA等）
   - [ ] 人名是英文名

### 第四步：测试40集（如果20集成功）
1. 重新创建项目，选择40集
2. 生成大纲
3. 应该分8批，每批5集
4. 检查是否全部40集都有内容

## 如果还是不行

### 问题1：人物生成失败
**现象**：点击AI生成角色后报错
**解决**：
- 检查是否先生成了创意（必须选择创意）
- 检查OpenRouter是否有额度

### 问题2：大纲只有前几集
**现象**：第3集开始没内容
**解决**：
- 降低集数到20集测试
- 检查浏览器控制台(F12)有没有红色错误
- 可能是AI偷懒，重新点击"重新生成大纲"

### 问题3：没有分批提示
**现象**：一直显示"AI正在生成..."
**解决**：
- 确保保存了 `js/ui.js`
- 强制刷新浏览器 `Ctrl+Shift+R`
