# 玛丽苏套路升级说明

## 🎀 核心改动：男强女弱玛丽苏设定

### 五大职业搭配（男强女弱）

1. **科技 Billionaire + 职场小透明**
   - 男主：科技 Billionaire（社交媒体/AI/App开发）
   - 女主：个人助理 / 前台 / 咖啡师

2. **华尔街金融巨鳄 + 职场新人**
   - 男主：投资银行总裁（冷酷并购之王）
   - 女主：实习生 / 数据录入员 / 食堂服务员

3. **地产帝王 + 设计菜鸟**
   - 男主：房地产大亨（掌控城市天际线）
   - 女主：设计师助理 / 中介新手 / 清洁女工

4. **时尚教父 + 时尚小白**
   - 男主：时尚品牌创始人（高冷毒舌）
   - 女主：造型师助理 / 试衣模特 / 店员

5. **酒店继承人 + 基层员工**
   - 男主：酒店集团继承人（玩世不恭贵公子）
   - 女主：前台 / 客房服务 / 实习厨师

### 玛丽苏人设要求

**男主角（强势霸总）：**
- 强势霸道，掌控全局
- 外冷内热，极度保护欲
- 事业巅峰的成功人士
- 冷酷高效，毒舌霸道

**女主角（柔弱小白花）：**
- 柔弱善良，有点笨拙
- 需要被保护，不强势
- 在基层岗位打拼
- 天真迷糊，软糯可爱

**关系动态：**
- 大灰狼 vs 小白兔
- 强强对比（男强女弱）
- 男主主导一切

### 🚫 女主职业禁忌

**不能出现的设定：**
- 高管、CEO、总裁
- 律师、医生、精英人士
- 女强人、创业者
- 比男主更有权势的角色

**必须是：**
- 助理、前台、实习生
- 服务员、店员、菜鸟
- 支持性、辅助性岗位
- 需要男主拯救和帮助
