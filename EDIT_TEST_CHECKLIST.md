# 编辑功能测试清单

## 🚀 快速测试（2分钟）

### 步骤1：生成创意
- [ ] 刷新页面 (Ctrl+F5)
- [ ] 点击"AI生成5个创意"
- [ ] 等待生成完成

### 步骤2：选择创意
- [ ] 点击第1个创意的"选择"按钮
- [ ] 确认下方出现"已选择的创意"区域
- [ ] 确认显示第1个创意的标题和内容

### 步骤3：编辑创意
- [ ] 点击第1个创意的"编辑"按钮
- [ ] 确认弹出编辑表单
- [ ] 确认所有字段都有内容（标题、核心设定、梗概、外在期待、内在期待）
- [ ] 在标题后面添加"【测试编辑】"
- [ ] 点击"保存修改"

### 步骤4：验证编辑结果
- [ ] 看到"✅ 创意已更新"通知
- [ ] 第1个创意显示"✏️已编辑"标记
- [ ] 下方"已选择的创意"区域显示修改后的标题
- [ ] 标题包含"【测试编辑】"字样

### 步骤5：推进到大纲
- [ ] 点击"进入角色设计 →"按钮
- [ ] 确认页面跳转到角色设计步骤
- [ ] 检查角色设计页面是否正常工作

## ✅ 预期结果

如果以上所有步骤都正常，说明编辑功能完全可用！

## ❌ 常见问题

### 问题1：点击编辑后表单为空
**解决**：检查控制台是否有错误，可能是idea对象没有正确传递到表单

### 问题2：保存后没有显示"已编辑"标记
**解决**：检查是否成功保存，查看控制台是否有错误信息

### 问题3：编辑后"已选择的创意"没有更新
**解决**：确认编辑的是已选中的创意，检查控制台日志是否有"[创意编辑] 已同步更新选中的创意"

### 问题4：进入角色设计后创意丢失
**解决**：检查第3步是否有错误，查看goToStep函数是否正确执行
