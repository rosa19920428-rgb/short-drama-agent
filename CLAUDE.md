## 核心功能模块

### 1. 密码保护系统

- **机制**: sessionStorage存储认证状态，关闭浏览器后失效

### 2. 创意生成
- AI基于类型生成10个创意
- 用户选择后进入下一步

### 3. 角色设计（AI生成详细小传）
- 点击"AI生成全部角色"
- AI基于创意生成5个详细角色
- 包含：外貌、性格、背景、动机、秘密
- 使用欧美名字（Sophia、Alexander等）

### 4. 分集大纲（分批生成）
- **每批5集**（避免AI偷懒）
- **中文输出** + **欧美场景** + **英文对话**
- 40集分8批生成
- 显示进度：(1/8批) → (2/8批)...

### 5. 完整剧本（逐集生成）
- **逐集调用AI**（1集1集生成）
- **好莱坞标准格式**:
  - FADE IN:
  - INT/EXT. LOCATION - TIME
  - 动作描述
  - 角色名（大写）
  - 对话（自然美式口语）
  - CUT TO: / SMASH CUT TO:
  - FADE OUT.
- **基于大纲扩展**（不是模板）
- 每集约300-500字（2页）
- 每集结尾有Hook

### 6. 女频浪漫喜剧元素（Rom-Com）
**针对Female → Billionaire CEO类别的特殊优化**:
- **意外亲密**: 摔倒、碰撞、意外接吻等物理接触桥段
- **嫉妒与占有欲**: 男主看到女主与他人亲密时的醋意表现
- **甜蜜惩罚**: 男主用亲密方式"惩罚"女主的调皮行为
- **被迫亲近**: 被困电梯、共用外套、只有一间房等设定
- **保护欲**: 危险时刻男主第一时间保护女主
- **反差萌**: 高冷霸总在女主面前展现温柔一面
- **秘密约会**: 伪装身份、隐瞒关系带来的刺激感
- **浪漫仪式**: 早安吻、晚安吻、告别吻等日常甜蜜

**AI生成要求**: 每集必须包含至少1-2个浪漫喜剧桥段，确保情感张力与趣味性

### 7. 剧本导出功能
**Word导出（好莱坞格式）**:
- 标准好莱坞剧本格式排版
- 元素层级:
  - 转场（FADE IN/OUT）: 右对齐
  - 场景标题（INT/EXT）: 左对齐，1.5英寸边距
  - 动作描述: 左对齐，1.5英寸边距
  - 角色名: 居中，3.7英寸位置
  - 括号（指示）: 左对齐，3.1英寸位置
  - 对话: 左对齐2.5英寸，右对齐2.25英寸
- A4页面设置，支持打印和编辑

**PDF导出**:
- 使用浏览器打印功能
- 自动应用好莱坞格式样式
- 支持打印为PDF文件

## 关键经验总结（2026-03-03更新）

### 创意生成与编辑功能

#### 1. AI生成创意必须有唯一ID
**问题**: AI返回的创意没有id字段，导致所有创意id为undefined，选择功能失效
**解决**: 在parseCreativeIdeas中为每个创意生成唯一ID:
```javascript
id: `idea-ai-${Date.now()}-${index}`
```

#### 2. 创意选择状态管理
**问题**: 重新生成创意后，旧的选择状态保留，导致显示混乱
**解决**: 生成新创意前重置选择状态:
```javascript
this.selectedIdea = null;
this.engine.project.selectedIdea = null;
this.engine.project.creative = null;
```

#### 3. 编辑功能同步机制
**问题**: 编辑创意后，已选中的创意不同步更新
**解决**: 编辑后检查是否已选中，如是则同步更新:
```javascript
if (this.selectedIdea?.id === ideaId) {
    this.selectedIdea = updatedIdea;
    this.engine.project.selectedIdea = updatedIdea;
    this.engine.project.creative = updatedIdea;
}
```

#### 4. 编辑状态标记
为编辑过的创意添加标记:
```javascript
isEdited: true // 显示✏️已编辑标记
```
